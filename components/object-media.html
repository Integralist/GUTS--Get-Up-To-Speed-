Responsive

A component can't be automatically made 'responsive'. This is the job of testers and developers to review how a component looks on different devices.

The tester will advise where a component's design appears to break and the developer will apply appropriate rules to correct the design at that break-point.

This same process has been applied below

So for example, when we set the display down to 320px wide the design breaks and so the developer implements a fix within the code, which you'll if it has been applied if you can see a red border around the images and they are set to be 50% of the screen width, making the text more readable.
